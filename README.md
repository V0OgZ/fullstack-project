### **📋 Prérequis**
```bash
Java 17+ (pour backend Spring Boot)
Node.js 16+ (pour outils développement)
Python 3.8+ (pour serveur interface port 8000)
Git (pour clonage et mises à jour)
```

### **🔧 Installation**
```bash
git clone https://github.com/V0OgZ/Heroes-of-Time.git
cd Heroes-of-Time
chmod +x hots
./hots start
```
